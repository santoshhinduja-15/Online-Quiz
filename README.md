# Online Quiz Web App  

A simple **MCQ-based quiz app** built with **HTML, CSS, Bootstrap, JavaScript, and jQuery**. Users can **answer multiple-choice questions**, see their **score in real-time**, and restart the quiz anytime.  

## Features  
✔️ Dynamic questions with multiple choices  
✔️ Real-time score calculation  
✔️ Responsive design (Bootstrap 5)  
✔️ Restart quiz option  
✔️ Smooth UI with jQuery  

## Technologies Used  
- **HTML5** for structure  
- **CSS3 & Bootstrap 5** for styling  
- **JavaScript & jQuery** for functionality  

## How to Run?  
1. Clone the repository:  
   git clone https://github.com/santoshhinduja-15/Online-Quiz.git
2. Open **index.html** file in browser
